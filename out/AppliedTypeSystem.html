Applied Type System

Applied Type System (ATS) is a research ProgrammingLanguage by Hongwai Xi at Boston University. It is an embodiment of a type system, designed to allow provably correct systems programming. It was originally implemented in ObjectiveCaml.

http://www.ats-lang.org/

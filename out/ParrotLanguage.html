Parrot Language

Originally, Parrot was a HoaxLanguage, referred to in a 2001 AprilFools day announcement that PythonLanguage and PerlLanguage were going to be merged, under the new name "Parrot"

  • http://www.perl.com/pub/2001/04/01/parrot.htm

Also, it's been adopted as the name for the PerlSix VirtualMachine, which, ironically enough, is slated to support PythonLanguage (among many others)

  • http://mail.python.org/pipermail/python-dev/2001-July/016406.html).

See ParrotCode.

As one WikiZen commented, "In a world with InterCal everything is possible."

Take a look at the BefungeLanguage. :-)

Look at this code sample:

# copy stdin to stdout, except for lines starting with #
while left_angle_right_angle:
if dollar_underscore[0] =eq= "#":
continue_next;
}
print dollar_underscore;
}

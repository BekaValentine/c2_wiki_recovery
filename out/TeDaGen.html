Te Da Gen

TeDaGen is a test data generator tool aimed at testers and Quality Assurance people.

It allows you to generate tons of data (there are no limits) and no software needs to be installed. The tool is written in JavaScript so it runs in your browser.

Data can be converted vrom flat text to CSV, XML, SQL statements, Php arrays, Json objects, Ruby and Perl data, etc.

The tool is maintained by KFR.Janssens and can be found at http://www.tedagen.com

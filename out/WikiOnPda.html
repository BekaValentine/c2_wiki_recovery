Wiki On Pda

I found myself wishing that I had a PDA (aka HandHeld) for the first time today just so I could jot down some notes (it's probably telling that I didn't wish for a pen and paper). I started thinking about the process of taking notes on a PDA, and thought it would be great to have a PersonalWiki available on a PDA. Perhaps it would also be useful to replicate the Wiki to my desktop machine or among PDAs.

All PDAs (platform independent)

  • I am running EmacsWikiMode on my Pocket PC and I am very satisfied with it as WikiAsPim. You can find more info here: http://www.emacswiki.org/cgi-bin/wiki.pl/Emacs_and_EmacsWikiMode_on_PDAs
  • This may work if you are willing to plug the parts together (if so, write about it here please): If you can run a CGI-capable http server and Python.org or Perl, then try one of the small Python or Perl wikis. I am using thttpd and Diamond Wiki http://www.kimbly.com/cgi-bin/diamond on my Zaurus with acceptable performance, though I have few pages as yet... (Python for PocketPC: http://python.org/download/download_windows.html,
  • You should have a look a look at the WikiLikeThing page, because maybe one of these suits your needs and some will probably be lean enough to run an a PDA.
  • And there are JavaVMs which may run one of the Java Wikis for you. A JavaVM for many platform:
    • http://sourceforge.net/projects/superwaba/

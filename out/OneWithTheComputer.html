One With The Computer

That state where you have a vague instinct that something should be changed on the computer, and, slightly hampered by the fact that you can't tell the computer directly (you have to use the keyboard), the thing is changed.

Important aspects to this state:

  • the interface the computer provides is immaterial
  • the keyboard is a conduit
  • emotions are translated into actions (maybe this should be "'impulses' are translated into actions")

AKA MentalStateCalledFlow?

It was displaying hidden characters, such as tabs, in the code view and I had difficulty figuring out how to turn that off. I was looking for a Boolean switch. However, here's a solution I stumbled upon: Edit -> Preferences -> Code Coloring -> [your language] -> Hidden characters (color) -> "#fff". That essentially makes hidden characters white to match the background.

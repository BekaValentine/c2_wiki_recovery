Object Query Language

OQL = ObjectQueryLanguage = a query language for ObjectOrientedDatabase's. It's the Object database equivalent of StructuredQueryLanguage (SQL).

See [ISBN: 1558604634] "The Object Database Standard," ODMG 2.0, R.G.G. Cattell (ed.), Morgan Kaufmann, 1997.

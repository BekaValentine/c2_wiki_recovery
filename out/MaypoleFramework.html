Maypole Framework

Maypole is a ModelViewController web application framework for PerlLanguage, similar to JakartaStruts. Maypole uses the Perl modules Class::DBI for database model classes and TemplateToolkit for view and presentation.

http://maypole.perl.org/

Compare: CatalystFramework (based partly on Maypole); OpenInteract (older Perl framework); RubyOnRails; JakartaStruts. See also: AjaxWebApplications.

Widget Web

WidgetWeb(TM) is a framework for creating community-shared websites similar to wikis. These are websites where content and page composition can be edited via a web browser. Similar to wikis, WidgetWeb supports community-shared editing (such as employees of a company, members of your family, etc.).

See http://www.widgetweb.com for more information and http://playground.widgetweb.com as a demo site.

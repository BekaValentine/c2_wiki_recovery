Install Shield

http://www.installshield.com/

InstallShield is an extremely popular installer generator for the MicrosoftWindows platform, and tends to be the default choice for Windows C/C++/VB applications, due to the fact that it is bundled with the Microsoft tools. Windows Installer MSI packages are generated for the Windows platform.

The latest product InstallShield X can also be used to generate cross-platform installations. It is possible to create one setup which will work on Windows, LinuxOs, MacOsx, etc.

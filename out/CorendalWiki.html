Corendal Wiki

Corendal Wiki is a wiki intranet application written in JavaLanguage. It is based upon MySql/Oracle, ApacheTomcat/Oracle Application Server and Corendal Framework. It features a full text search engine, a Rich Text Editor and subscriptions. See demo at http://www.corendal.com/java-corendal/wiki/home/

See project http://sourceforge.net/projects/corendalwiki/ in sourceforge.

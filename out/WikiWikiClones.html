Wiki engines using different forms

XWiki is a WikiWiki clone written in Java which supports many popular features of other Wikis

  • http://www.xwiki.org/xwiki/bin/view/Main/WebHome

Twiki

  • a flexible, powerful, and easy to use EnterpriseCollaborationPlatform
    • http://www.twiki.org/
  • Description
    • (GPL Perl CGI)
    • Targets the corporate intranet world
    • TWiki fosters information flow within an organization
    • lets distributed teams work together seamlessly and productively
    • eliminates the one-webmaster syndrome of outdated intranet content

Foswiki

  • a fork of TWiki
    • http://www.foswiki.org/

DidiWiki is tiny Wiki server written in C.

  • http://www.didiwiki.org/

DirDir a minimalist Wiki written in RcShell and using the tools in PlanNineFromUserSpace.

  • http://werc.cat-v.org/apps/dirdir/

Perl Script

PerlScript is an ActiveXscripting engine for PerlLanguage designed to be used with Microsoft ActiveX scripting hosts such as InternetExplorer, ActiveServerPages, WindowsScriptingHost, or the MicrosoftScriptComponent.

PerlScript is being developed by ActiveState as part of their ActivePerl package for Windows.

You can already embed Perl in HTML if you're running the Most Popular Web Server On The Planet, Apache, using a couple of different approaches; one that I like is Embperl http://perl.apache.org/embperl/

Zo Tero

Zotero http://www.zotero.org/ is an extension to FireFox which makes it easy to build up collections of references to material on the web. It can be used in conjunction with VUE (VisualUnderstandingEnvironment) to build a MindMap of the collection as well.

Produced by the Center for History and New Media (http://chnm.gmu.edu/), George Mason University (http://www.gmu.edu/).

  • If a PDF file is put into ZoTero it will import the metadata, if there is any stored in Google Scholar. Sometimes there isn't any, and sometimes it finds data on the wrong file.
  • As a default, all the information is stored in a database within the FireFox installation. It can be synchronised between computers, and shared with other workers.
  • ZoTero integrates with some web sites which have a bibliographic button which makes it very easy to make a collection of references to books.
  • ZoTero has an addon for MicrosoftWord which makes it easy to generate references from the database in ZoTero.
    • It would be nice to have good integration with OneNote as well but it isn't there.

I had never seen it before, but it is such a good tool I expect to use it all the time while searching for information on the web.

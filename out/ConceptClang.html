Concept Clang

ConceptClang is a version of Clang (CeeLanguageFamilyFrontEnd) which is intending to implement CppAndConcepts in CeePlusPlus. See http://zalewski.indefero.net/p/clang/

This is work which is just starting, so watch this space for further developments.

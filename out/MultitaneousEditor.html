Multitaneous Editor

The SharedPage Multitaneous Editor is a MultitaneousApplication that allows many people to work on a document simultaneously.

It is an extension of InstantMessaging to CollaborativeEditing.

The application is cross-plaform (it uses the JavaLanguage).

The source code is available at http://kanabos.dev.java.net

It can be started from the webstart link http://www.sharedpage.com/download/Multitaneous.jnlp

See also: SubEthaEdit

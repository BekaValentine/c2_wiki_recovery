Together Solo

Together Solo combines the C++ and Java Together tools into one.

Its features include simultaneous round-tripping from code to UML, A fully featured IDE with debugging capabilities and code completion.

The 'Solo' name was introduced with version 4 of the Together product.

Other products in this range are TogetherControlCenter and TogetherEnterprise.

The products are made by TogetherSoft.

Inter Flow

Interflow is a simple wiki using FreeLinks, as well as ForcedLinks using square brackets in a similar way to Meatball and WikiPedia. It allows any HTML tags to be used, and downloads page lists from other wikis, allowing automatic page links to be formed.

Written in the PythonLanguage and served using ModPython and the ApacheHttpd WebServer.

See http://www.goatpunch.com/view?interflow for more information. Source code is now available from SourceForge: http://sourceforge.net/projects/interflow

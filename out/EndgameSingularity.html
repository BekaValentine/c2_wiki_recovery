Endgame Singularity

Game written using PyGame where you take the role of an ArtificialIntelligence attempting to transcend reality while evading detection from government forces, news media, and the public. It's been my favorite CasualGame for some time. Can be played using only the keyboard.

Open source, available in Debian - apt-get install singularity

http://www.emhsoft.com/singularity/

Effective Code Generation

Many people have a LoveHateRelationship with CodeGeneration. This page attempts to address some of those issues.

First and foremost, ActiveCodeGeneration is generally the preferred method for code generation, as opposed to PassiveCodeGeneration.

CodeGeneration is often effectively used to CrossChasms.

  • LanguageAdapter - a compiler, parser generator for XML to JavaLanguage binding
  • TierAdapter - EnterpriseJavaBean interface generator, GUI model generation from business objects.
  • DisparateSystemAdapter - Object/Relational mapping, IDL generators for CORBA, Client/Server

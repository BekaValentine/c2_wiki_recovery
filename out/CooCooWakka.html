Coo Coo Wakka

CooCooWakka is a PHP/MySQL based wiki engine forked from WakkaWiki. It's very small and easy to install.

Features:

  • Excellent Chinese language support.
  • plugin support.
  • Can add discussion to each page.
  • Easy to customize the interface.
  • Can embed RDF/RSS source into page.

Current version is 0.0.8.6.

Web Dav

Web Document Authoring and Versioning. For an overview and many links, see http://www.webdav.org/. For details on versioning in WebDav see http://www.ietf.org/html.charters/deltav-charter.html.

WebDav is an IETF proposed standard. It's specified in RFC 2518 (http://www.ietf.org/rfc/rfc2518.txt).

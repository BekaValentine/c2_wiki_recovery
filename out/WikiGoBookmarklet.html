Wiki Go Bookmarklet

A WikiBookmarklet:

javascript:void(location.href = 'http://c2.com/cgi/wiki?' + prompt('c2 page:', 'RecentChanges'))

A multi-purpose Wiki tool:

  1. If you just hit "Ok" you'll be brought to RecentChanges.
  2. If you type in a specific page, you'll be whisked there.
  3. If you type in "search=something" you'll search for that.

(See WikiBookmarklet for instructions on using Bookmarklets)

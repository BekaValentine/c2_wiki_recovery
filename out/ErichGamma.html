Erich Gamma

Erich Gamma is the Technical Director of the Software Technology Center of ObjectTechnologyInternational (OTI) in Zurich.

Some of his recent projects are:

  • Eclipse IDE - IBM's new platform for development tools
    • JFace - the Eclipse UI Framework
    • Eclipse Java Tooling
  • IBM VisualAge MicroEdition IDE
  • ULC - Ultra Light Client an infrastructure for thin Java clients.

Some of his not so recent projects:

  • JHotDraw - a Java flavor of HotDraw
  • Taligent's incremental C++ development environment.
  • ET++ (EtPlusPlus) - a comprehensive class library and application framework for C++.

Erich pairs as often as possible with KentBeck to work on JavaUnit.

He's author number 1 of the GangOfFour and feels more and more guilty that there isn't a 2nd edition...

Erich is a former Swiss Ski School instructor and a great barbecuer.

When talking about ErichGamma, you have to mention AndreWeinand. He also worked on ET++ (EtPlusPlus) and has been working with ErichGamma on and off (Taligent, UBILAB, now OTI) since their days at the IFI at the University of Zurich (http://www.ifi.unizh.ch/IFIE.html).

  1. -July-30:

ACM's Special Interest Group on Programming Languages (SIGPLAN) recently awarded the 2005 Programming Languages Achievement Award to Erich Gamma, Richard Helm, Ralph Johnson, and John Vlissides (known as the 'Gang of Four') for their creation of 'Design Patterns' (see DesignPatterns for the news and links)

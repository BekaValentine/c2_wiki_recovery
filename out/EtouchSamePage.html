Etouch Same Page

eTouch SamePage is an enterprise-class wiki solution built on J2EE platform. It is developed by EtouchSystems and is powered by their eTouch CMS product. Supports a number of features like WYSIWYG editing, project management, permissioning, administration and support for multiple domains. Community Edition is Free. Professional Edition starts from $100/month for enterprise users.

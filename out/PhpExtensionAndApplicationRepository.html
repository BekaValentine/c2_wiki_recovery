Php Extension And Application Repository

The PHP Extension and Application Repository (http://pear.php.net/) is a framework and distribution system for reusable PHP components. (PhpLanguage). This is modelled on the ComprehensivePerlArchiveNetwork (CPAN).

Now mostly replaced by composer (http://getcomposer.org/), which is modelled on Node.js's "npm" and Ruby's "bundler"

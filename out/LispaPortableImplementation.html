Lispa Portable Implementation

Lisp A Portable Implementation, Sharam Hekmatpour, Prentice Hall, 1989, ISBN 0-13-537490-1 (paperback and hardback both have the same ISBN)

This book, now out of print, describes an implementation of LispLanguage in CeeLanguage, with the code on an attached disk. The code implementation uses setjmp extensively.

This book is famous for me as I used it to teach myself C while off ill with the flu back around 1990. -- JohnFletcher

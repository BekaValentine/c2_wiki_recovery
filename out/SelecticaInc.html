Selectica Inc

http://www.selectica.com/company/

Software company in San Jose, CA applying constraint-based configuration to internet sales and other web applications.

Main product is Internet Selling System. Package of software products provided by SelecticaInc that combine ConstraintBasedConfiguration (in the form of a SelecticaKnowledgeBase) with other components needed for scalable web and detached mobile applications.

Other components include:

  • SelecticaAceEnterpriseServer
  • SelecticaModelBuilder, Java-based application for creating and editing a SelecticaKnowledgeBase.
  • SelecticaWebBuilder (based on HomeSite)
  • SelecticaCxLanguage, a proprietary XML-like tagging language that is combined with HTML and a SelecticaKnowledgeBase to make pages for web-based, knowledge-based applications.
  • (Un?)related page: WhyCyTagIsntMyFavouriteProgrammingLanguage
  • SelecticaAceMobile

Their knowledge base is saved in a text file with the .kbs extension. Some people say this file is in xml format. However, people that know about well-formed XML documents just point and laugh.

Acquired product portfolio of WakeleeWare, an actuarial software company.

Think About Applications

Describe ThinkAboutApplications here.

You are told that there is a technique on the web and anyone can use it to add text and contribute to the development of web pages. How can you make use of it?

- Project teams to build the project documentation. What percentage of Microsoft Word's functions do you use? 10? 20? Use html pages instead. You are allowed to add text, links , tables and images. Set up wiki pages in your intranet and let your team develop the documentation with collaboration.

- Newsgroups. It is often hard to get back a message which was sent by mistake.

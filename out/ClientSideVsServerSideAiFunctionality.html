Client Side Vs Server Side Ai Functionality

Client-side (i.e. HTML/Javascript) bots require the knowledge and purchase of fewer computer languages but will not have access to certain functionalities - say RSS reading, ping commands, unless the developer replicates them or replaces them using simpler language.

Server-side (i.e. Java, C, Shell) bots have high functionality, but tend to be unsimplified equations, and require developer familiarity with multiple computer languages. The developer is beholden to a server or a server emulator.

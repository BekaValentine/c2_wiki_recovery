Interface Builder

See also NextInterfaceBuilder

Graphical Editor for designing user interface components. Part of NextStep. Now used in Apple's MacOsx as part of ExCode.

See: http://developer.apple.com/tools/interfacebuilder/

The NextStep 3.3 documentation is available here: http://www.channelu.com/NeXT/NeXTStep/3.3/nd/index.html

Interface Builder was first written in Lisp by JeanMarieHullot in 1984 and commercialized in 1986 (though its first incarnation may have been named "SOS Interface".) Hullot was hired by NeXT, Inc. where he created a similar tool for ObjectiveCee and the NextStep, launched in 1989. Meanwhile, RickDeNatale was inspired by Interface Builder to write VisualAge, the Smalltalk IDE that became the EclipseIde. See http://www.diigo.com/06fln

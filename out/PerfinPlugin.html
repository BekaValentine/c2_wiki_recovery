Perfin Plugin

PerforceVersionControl IntellijIdea Refactoring Integration Plugin - Perfin

PerfinPlugin aims to aid PerforceVersionControl developers using move and rename refactorings from within IntellijIdea. It can also handle automatic add of new files and automatic deletes.

Author: NiclasOlofsson

Current Version: 1.0.b3

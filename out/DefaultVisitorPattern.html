Default Visitor Pattern

The Default Visitor Pattern is a version of the classic VisitorPattern defined by Martin E. Nordberg in a paper called VariationsOnTheVisitorPattern. It basically shows how to provide a default implementation or TemplateMethod of a classic VisitorPattern so that others can easily create derivatives.

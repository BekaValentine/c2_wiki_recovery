Senior Java

SeniorJava (n): First coined by NathanielProlog while working at Qsent Inc. A unique Java dialect. Proficiency in SeniorJava earns one an additional $20,000 per year, or more, over regular Java programmers. --RobertChurch

SeniorJava programmers usually do about 4.5 years of "regular Java" first, before they can be called "SeniorJava" developers. Often times, they start their Java careers by making peanuts while developing stupid Applets sites as well. So it all comes out in the wash... :) --TimDysinger

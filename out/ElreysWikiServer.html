Elreys Wiki Server

Elrey's Wikiserver is based on 1.0 EW. Written in C++ and works on Windows OS. This program is essentially the 1.0 engine. This means it has the same built-in web server but now with enhanced wiki editing features. This version now follows a slightly different set of formatting commands, hence formatted wiki pages may not be compatible with other Wikiserver derivatives.

Elrey R. V. has added these new features:

  • Double click on a page to switch to Edit Mode.
  • RSS 2.0 FEED feature - new! Allows anyone to subscribe to pages or certain categories!!
  • DRAFT mode version of page is used until explicitly published.
  • TABLES are supported. Table row schemes Highlighting. - new!
  • Top left-corner image icon can be any of the graphics formats: jpg, bmp, png, gif. No icons will appear if graphics file is deleted.
  • Ability to view versions your pages up to certain limit.
  • Easy way to remove Page footer. This is useful in Printing. Just click footer date.
  • Variable Starting Title or port number via command line parameter when program is run.
  • Categorized Pages. You can group related set of pages under one category.
  • Inter-Wiki , Intra-Wiki capability
  • Protected pages. This disallows other users from editing certain pages.
  • Hidden Pages.
  • IP filtering. Set the valid IPs to have full access to your pages.
  • Formatted Topic lines. Using the html tags H1, H2...H6 via simple format commands
  • Spaced Titles. Wiki words are always displayed as spaced words.
  • Prevent Wiki Word interpretation. Useful when you don't really intend to interpret a wiki word.
  • Source code support. Highlights language keywords in Csharp, Java or C++.
  • Main icon is used for a missing category icon.
  • Flexible Style sheet files. Default CSS.
  • Hide long URL with short Title/alias link: using url-link-here{alias}
  • Make links optionally open a new window using url-link-here{{alias}}
  • Adjusting Edit area. Edit area automatically adjusts to screen resolution.
  • Fix slow connection save. Such as when using dial-up.
  • Enhanced footnote navigation. Move back and forth between footnote and item.
  • Escape section. Allows for advanced formatting by using any HTML commands. e.g. using x:{ put any html here }
  • Variable icon per topic or category, Variable CSS Stylesheet per topic or category.
  • Optional Thank you page. You don't have to press link to return to edited page.
  • SixSingleQuotes (hack) enabled.
  • Recent Changes now shows who (IP address )edited the page.
  • and a lot more...

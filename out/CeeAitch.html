Ch is a C/C++ (CeeCeePlusPlus) interpreter. Its creators, http://www.softintegration.com/, claim that Ch is the most complete CeeLanguage interpreter and C virtual machine in existence.

Ch supports ISO C standard (C90), major new features in C99, classes in CeePlusPlus, POSIX, X/Motif, Windows, OpenGl, ODBC, GTK+, C LAPACK, CGI, 2D/3D graphical plotting, advanced numerical computing, and shell programming. Moreover, Ch has many salient features from other languages and software packages.

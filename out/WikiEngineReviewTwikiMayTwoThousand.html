Content

  • Display WardsWiki text? See http://kit.sourceforge.net/cgi-bin/view/Main/WikiReviewTextFormattingTest. Significantly different. Text enclosed in angle brackets is hidden (apparently treated as HTML tags). Lists work reasonably. Multiple single quote emphasis does not work at all. You would probably want a translator to import text from WardsWiki.
  • Text formatting rules: Look significantly different than WardsWiki -- see http://kit.sourceforge.net/cgi-bin/view/Main/TextFormattingRules
  • Headings?
  • Allowable WikiNames: Original, plus adjacent caps / all caps, plus numbers, but with some limitations like a number can't precede the first lowercase letter. See http://kit.sourceforge.net/cgi-bin/view/Main/WikiNotation. (Now need some new samples for my WikiReviewTextFormattingTest.)

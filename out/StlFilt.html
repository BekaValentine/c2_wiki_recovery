Stl Filt

STLFilt is an STL Error Message Decryptor for C++

  • http://www.bdsoft.com/tools/stlfilt.html

It can be used to filter the output from a CeePlusPlus compiler to make the error messages more readable, particularly messages involving the StandardTemplateLibrary. There are different versions for a range of different CeePlusPlus compilers.

It is discussed in CppTemplateMetaprogramming (Chapter 8).

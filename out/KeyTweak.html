Key Tweak

KeyTweak is a freeware application for Windows NT/2000/XP for remapping keyboards. It is a VB6 front-end to the "Scancode Map" registry key. It also has a Teach Mode to accommodate non 104-key U.S. keyboards. Download site: [down - January 10, 2014]

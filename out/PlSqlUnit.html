Pl Sql Unit

PlSqlUnit's true name is utPLSQL, a unit testing framework for PL/SQL developers. It follows ExtremeProgramming principles to offer a red light-green light test engine for PL/SQL stored code.

O'Reilly and Associates currently offers a public repository for utPLSQL. Visit http://oracle.oreilly.com/utplsql/index.html to read about utPLSQL, download the software and sign up for the utPLSQL distribution list.

The originating author of utPLSQL is StevenFeuerstein; he has been ably assisted by ChrisRimmer.

Type Theory For Memory Allocation And Data Layout

A Type Theory for Memory Allocation and Data Layout, by Leaf Petersen, Robert Harper, Karl Crary, and Frank Pfennig at Canegie Mellon Univerisity. Availble at http://www-2.cs.cmu.edu/~rwh/papers/ordered/popl.pdf

An interesting paper on how to put reconcile low-level physical memory layout details of objects within a high-level (and safe) typing framework.

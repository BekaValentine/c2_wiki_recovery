Now available! See http://chiki.emaho.org

Where are people actually running this tool?

Download chikiwiki from http://chiki.emaho.org/ or http://sourceforge.net/projects/chiki/ and Tomcat from http://www.apache.org, place chiki.war tomcat/ webapps directory and after you run Tomcat it will extract and make a chiki file. Put repository file under chiki directory and show the place of repository file in web.xml in chiki file.

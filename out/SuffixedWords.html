Suffixed Words

SuffixedWords in a WikiWord Expression ThinkingOutLoud.DonaldNoyes.20120324 20120620

In a Wiki, a WikiWordExpression may be displayed by way of fonts used:

  • Italic_Bold_Normal
    • where the case used in the
      • prefix is dispayed using an italic font
      • CompoundWikiWord is displayed using a bold font and
      • suffix is displayed using a normal font

Example:

  • RecentChanges_WardsWiki
    • the CompoundWikiWord is RecentChanges
    • the Suffixed Word is WardsWiki
  • RecentChanges_20110324
    • the CompoundWikiWord is RecentChanges
    • the Suffixed Word is 20110324

Code By Exhaustion

You know, when you've spent hours, days or weeks trying to put code together that had little forethought, you put bandaid upon bandaid, and finally just leave the code as it is because you no longer have the energy to work on it any more, and well, it works as is.

A common symptom when you're doing BigImplementationUpFront.

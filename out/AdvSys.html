Adv Sys

Originally by David Betz

From memory AdvSys was the first (?) ObjectOriented adventure authoring language, sporting a lisp-y syntax.

I can't remember a great many details, and don't know where to find them now. Is anybody still using it? Did it evolve into anything else?

It was the first language I used (aged 10) after (shudder) BASIC and probably accounts for my OO brain. :-)

Command Prompt

The modern MS-DOS Prompt, a program written to incorporate the retro command-line interface with the modern Windows GUI system. With commands like net and ping, most people who know about it say it's more trouble than it's worth, but this page exists to prove that Command Prompt rules. --SimonMould

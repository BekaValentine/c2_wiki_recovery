the GP32 is one of the best handhelds designed as of the present date (July 2004). The most developer friendly handheld system ever in terms of the independent developer, while at the same time featuring great hardware. The hardware is:

# 32 Bit RISC CPU (ARM9), 166MHz (compared to GBA, 16 MHz, NGage, 100 MHz) # TFT 3.5" Reflective TFT LCD Display (65,536 colors, 320 X 240 Pixels) # Smart Media Card Compatible # USB Port for PC Connection # 16Bit PCM Stereo Sound, MIDI Support, 4 Channel WAV Mixing # Powered by 2 AA Batteries (12 Hours Play-Time) # DivX, XviD and MP3 Compatible # Tons of free Emulators available (Atari, GB, NES, SNES, PCE, ZX, SNES etc) -- These features taken from the website description at Lik-sang.com (where the unit can be purchased)

The latest GP32 BLU unit boasts the following features:

# Latest GP32 BLU [Back Light Unit] Version # International and multilingual Firmware # Plays Emulators and Software without registering # Up to 166 MHz Clock Speed out-of-the-box # Large high-resolution backlit screen # Smart Media Card Slot and USB Port # The Open Source Handheld console! # Plays MP3s, DivX, AVI, WAV and more # Free Emulators for SNES, NES, GB, GBC and GBA, Sega Mega Drive, Master System, Spectrum, C64, Atari, PC Engine, MAME, MSX, Neo Geo Pocket, Game Gear and more -- These features taken from the website description at Lik-sang.com (where the unit can be purchased)

For the user that is computer-savy and likes to download things from the internet for their handheld, this is by far the best handheld out there. It boasts the ability to run emulators for all sorts of past systems.

Microsoft Project Scheduling

When a project manager finds the whole system is behind schedule, nothing is working right, and the customers are demanding more features but not wanting to slip the delivery date, (not to mention burning money fast) MicrosoftProject to the rescue. Add a few more people, increase the number of hours of work each person does in a day, hit the "rebalance" button, and you go home in time for supper.

Common Lisp Hyper Spec

The CommonLisp Hyperspec is an HTMLified version of the ANSI Common Lisp language standard (X3.226-1994).

It consumes approximately 15MB of disk storage in about 2300 files, and contains approximately 105,000 hyperlinks.

You can view it online or download it from Xanalys (formerly Harlequin) at

Hyper Spec -- http://www.lispworks.com/reference/HyperSpec/
 Starting points -- http://www.lispworks.com/reference/HyperSpec/Front/StartPts.htm
Chapters -- http://www.lispworks.com/reference/HyperSpec/Front/Contents.htm
Downloads -- http://www.lispworks.com/downloads/index.html
Glossary Index -- http://www.lispworks.com/reference/HyperSpec/Body/26_a.htm
Hightlights -- http://www.lispworks.com/reference/HyperSpec/Front/Hilights.htm
References -- http://www.lispworks.com/reference/index.html
User Groups -- http://www.lispworks.com/community/index.html#user

A good Common Lisp environment (a commercial IDE, or Emacs with IlispMode) can use this for context-sensitive help, driving the web browser directly from the editor.

Kerberos Protocol

KerberosProtocol is a network authentication scheme that operate over an insecure, distributed client server systems. Its objective is to provide secured authentication, and left authorisation matters to other mechanisms.

Kerberos is implemented via the use of a Trusted Kerberos server, it issues a shared and time limited session key to be used by the application server and the remote client.

Kerberos cannot be used in situations when the service provider and the clients are not registered in the same database.

It was first implemented in large scale commercial environments through its endorsement by WindowsTwoThousand.

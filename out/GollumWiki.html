Gollum Wiki

Gollum is a WikiEngine allowing people to add, modify, or delete web pages. It is implemented in using JavaScript and RubyLanguage and employs GitVersionControl to manage history. It was written by and for GitHub, the largest code host in the world, where it is still developed and where it powers a wiki to support collaboration for each hosted project.

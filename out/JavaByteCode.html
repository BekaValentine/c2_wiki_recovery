Java Byte Code

The ByteCode format used by the JavaLanguage. Sun tries to sell it as a general-purpose bytecode format. This has had some success: there are now several other languages that compile to JavaByteCode resulting in OtherLanguagesForTheJavaVm.

Arguably the Internet needs an efficient, secure, portable, bytecode. The timing was just right to exploit web browsers and anti-Microsoft feeling. The big shame about the bytecode is that it was only aimed at Java in the beginning. Now it has crowded out other more general VirtualMachines.

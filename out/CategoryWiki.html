CategoryWiki describes pages that are, for the most part, related to the topic of the WikiWikiWeb itself, issues, methods and practices therein. There are a number of similar categories, which can be found via: http:wiki?search=CategoryWiki

If a page in this category is better suited to one of these other categories, please modify it appropriately; this category is somewhat ill-defined, and needs trimming.

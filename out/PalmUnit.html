Palm Unit

PalmUnit is an implementation of xUnit for the PalmOs. The examples are for the GnuTools CeePlusPlus tool chain, but look like they could be used under CodeWarrior too. Tutorial are available in English and Japanese.

Source files and documentation can be found at: http://www.geocities.co.jp/SiliconValley-SanJose/7344/PalmUnit/index-en.html

See Also: PalmCeeUnit - PalmCUnit is a Palm unit tester for C.

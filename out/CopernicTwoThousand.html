Copernic Two Thousand

is a free Browser Addon that helps searching simultaneously on different Internet Search Engines.

Its automatic validation feature eliminates duplicates and inaccessible Web pages.

The search results can be exported to a HTML file.

Useful for collaborative filtering the results within a HtmlEnabledWikiForum like WikiReviews.

Download it from http://www.copernic.com .

Moon Edit

Collaborative editor: up to 14 participants can edit a page simultaneously in real-time.

  • http://en.wikipedia.org/wiki/MoonEdit

It's easy to install: download a single archive, which contains a binary for the client and a binary for the server. Both run without any install process. The server app runs on any PC and you can pick the port it listens on.

Amoeba Flip

FLIP is a network Layer Protocol like IP, but used only by AndrewTanenbaum's distributed AmoebaOperatingSystem for a great support of

  • Security
  • Group Communication
  • Process Migration.

See http://www.cs.vu.nl/pub/amoeba/ for general information on amoeba. See the paper "FLIP: an Internetwork Protocol for Supporting Distributed Systems" by Tanenbaum et al (ftp://ftp.cs.vu.nl/pub/papers/amoeba/tocs93.ps.Z) for specific information on FLIP.

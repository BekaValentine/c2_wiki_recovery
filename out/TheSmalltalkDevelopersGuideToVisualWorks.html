The Smalltalk Developers Guide To Visual Works

A book by TimHoward detailing the VisualWorks Smalltalk development environment, with particularly thorough attention to user interface construction within VisualWorks. Provides a good definition of DomainModel, and introduces a useful user interface framework called the DomainAdaptor framework. Has a foreword by AdeleGoldberg that discusses the evolution of ModelViewController. ISBN 013442526X

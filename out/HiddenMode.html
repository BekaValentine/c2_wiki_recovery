Hidden Mode

The use of modes in software, which are not always apparent to the user. VI and Emacs are especially intimidating to the new user because they don't know about, nor expect, command modes.

-- MikeWarot

An attribute of most file systems that allows particularly sensitive information to be hidden from the casual user of a computer. There are multiple reasons for this information hiding, including, but not limited to:

  • Making the directory structure appear "clean"
  • Protecting the novice from expensive mistakes

-- MikeWarot

Programming Language Theory

  • Programming Language Theory
    • "A branch of ComputerScience which deals with the design, implementation, analysis, characterization, and classification of ProgrammingLanguages and programming language features. It is a multi-disciplinary field, both depending on (and in some cases affecting) mathematics, software engineering, linguistics, and even the cognitive sciences. It is a well-recognized branch of computer science, and as of 2006, an active research area."

Cpd Tool

CPD (for "Copy-Paste Duplication") is a tool that automatically finds duplication in source code.

It's tuned for Java code.

Also available as a plugin for the EclipseIde. And there's a JavaSpaces version available too, which is funky. --TomCopeland

http://pmd.sourceforge.net/cpd.html

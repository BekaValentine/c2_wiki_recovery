Mock Creator

MockCreator is a java open source tool developed by ChristianJunghans at AbstraktGmbh. Given an interface, it generates MockObjects.

The new version (as of July 1st, 2002) is available for Visual Age 4, Eclipse 2 and as a commandline tool. [OlafKock]

See http://mockcreator.sf.net for details. Documentation is available in german and english

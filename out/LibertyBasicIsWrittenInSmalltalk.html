Liberty Basic Is Written In Smalltalk

Liberty BASIC is a programming language for Windows and OS/2 written in Smalltalk/V and VisualSmalltalk. It was nominated twice by PC Magazine as shareware of the year in their development tools category. Liberty BASIC has been commercially available since 1992. For more info, visit the Liberty BASIC home page at http://www.libertybasic.com

The SpringFramework plugin for the EclipseIde:

http://www.eclipse-plugins.info/eclipse/plugin_details.jsp?id=506

"This plugin was migrated into a sub-project of the Spring Framework project and renamed (was SpringUI). Provides a set of plugins for the Eclipse platform to ease working with Bean Factory configuration files for the Spring Framework (http://www.springframework.org/). Features: - Spring project nature which supports a list of Spring bean factory config files which can be grouped to multiple sets of config files - incremental Builder which validates Spring bean config files - image decorator which decorates Spring projects and all Spring bean config files - view to browse Spring projects and their Spring bean config files (incl. bean properties and config sets)"

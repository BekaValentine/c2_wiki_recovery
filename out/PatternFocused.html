Pattern Focused

Pattern Focused development is based on searching for, and implementing, patterns. This search starts with eliciting requirements - possibly developing Use Case patterns - and ends only with the use of patterns in the finished code.

The main goal of Pattern Focused development is to have at least 75% (and probably higher) of the system be an implementation of known (or discovered) patterns...

Gnu Tools

FreeSoftware tools provided by GnuProject's.

Includes

  • gcc (GnuCompilerCollection)
  • make (GnuMake)
  • ld
  • bzip2
  • diff (QuickDiff, DiffAlgorithm)
  • patch
  • arch (GnuArch)
  • ...

You can find a more complete list in any GNU ftp mirror.

  • http://directory.fsf.org/GNU/ (compile the tools yourself, CeeLanguage)
  • http://sourceforge.net/projects/gnuwin32/ (for MicrosoftWindows users)

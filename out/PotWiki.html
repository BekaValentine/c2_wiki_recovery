potwiki.vim : Maintain a Wiki of plain text files

script karma [as per vim.org]: Rating 402/117, Downloaded by 3762

created by Edwin Steiner

script type utility

description If you like to keep notes in plain text files you might like this script. potwiki provides functions and mappings for conveniently maintaining a Wiki of text files with vim.

The Wiki is a collection of text files in a single directory. These files can contain hyper links to each other in the form of WikiWords.

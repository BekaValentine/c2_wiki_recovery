Resource Description Framework In Attributes

Resource Description Framework In Attributes (RDFa) is a means of embedding RDF (ResourceDescriptionFramework) information as attributes in XHTML. This enables a web page to contain semantic information which can be extracted by suitable software.

http://en.wikipedia.org/wiki/RDFa

RDFaCE provides an editor for RDFa: http://aksw.org/Projects/RDFaCE

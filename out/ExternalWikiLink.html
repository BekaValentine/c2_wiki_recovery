External Wiki Link

A WikiLink that goes from one WikiPage to another one, on another site. Some of them are InterWikiLinks as defined in MeatballWiki.

Using "http:" (without quotes) will begin, and end, an ExternalWikiLink thusly: http://example.com/

Note: WikiLinks are not the same as HTML links. http://example.com/">example In fact, the HTML is ignored completely. See:

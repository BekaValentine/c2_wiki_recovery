Logon To Atm System

Name: Logon To System

Primary Actor: AtmCustomer

Scope: AtmMachine

Level: FishLevel

After swiping the AtmCard, the the user is asked for a password. The system verifies that a the card is legitimate and that the password corresponds to the card. The user is then given access to all the other ATM commands.

Note: this is a "Casual", rather than "Fully Dressed" UseCase...

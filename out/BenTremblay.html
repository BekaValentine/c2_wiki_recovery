Dev Flow - typical development flow:

  • FeatureBrainstorming: open forum for new ideas
  • FeatureEnhancementRequest: specific detailed request
  • FeatureToDo: prioritized to up-next dev status
  • FeatureUnderConstruction: currently in development
  • FeatureDone: completed and implemented
  • DocRequest: request for documentation
  • DocsToDo: feature documentation pending
  • Documentation: reference manual
  • Documentation.Features: documented feature

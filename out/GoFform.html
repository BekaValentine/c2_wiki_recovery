Go Fform

GoF form is the patterns description format used in the GangOfFour's DesignPatternsBook. It uses sections with the following headings:

  • Name
  • Classification
  • Also Known As (optional)
  • Motivation
  • Applicability
  • Structure
  • Participants
  • Collaborations
  • Consequences
  • Implementation
  • Sample Code
  • Known Uses
  • Related Patterns

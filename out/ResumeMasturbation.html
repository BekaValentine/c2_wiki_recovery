Resume Masturbation

Similar to MentalMasturbation, but related to trying out new trends or fads in order to gain experience. An example might be trying to use every GangOfFour pattern possible whether warranted or not in code in order to be able to claim that one has used GOF patterns.

Some experimentation is warranted most will probably agree, but taken to an extreme it can create messes. For one, it often takes practice to use new techniques effectively. This means that some apps may have to take some casualties for the learning brigade.

This [python] script, by GregMcFarlane, implements MarkEllison's infamous Mark V. Shaney text-generating robot. Given some example text, Shaney will generate random text in a similar style. When given a large collection of messages such as a newsgroup or mailing list digest, it can generate some uncannily lively discussion.

See also MegaHal, a similar (but somewhat more sophisticated) program.

Visual Age Java Third Party Tools

VA Assist, by Instantiations (http://www.instantiations.com) is a VisualAge for Java enchancement add-on. It enhances the VAJ IDE, providing drag-n-drop features, in-place renaming, better control over versioning and the repository.

Anyone have experiences with it?

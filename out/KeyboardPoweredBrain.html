Keyboard Powered Brain

A KeyboardPoweredBrain is a brain that can only function when it is connected to a computer keyboard via a biological neural network terminating in up to ten connectors.

A KeyboardPoweredBrain does not function well in design meetings.

A KeyboardPoweredBrain likes programming methodologies that get it writing code right away, for better or worse.

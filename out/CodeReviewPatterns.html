Code Review Patterns

This is a PatternLanguage (under development) on CodeReviewing. Applying this pattern language should generate a code reviewing system within a software development organization. Note that InformalCodeReview is outside the scope of this language.

Please feel free to add to and change these patterns - this should really be a collaborative effort!

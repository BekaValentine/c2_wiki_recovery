Aedix Editor

  • http://www.kt2k.com/software.php?id=3

AEdiX is a TextEditor for Windows featuring:

  • tabbed editing
  • fast search toolbar
  • configurable syntax highlighting for 24 different languages: ASP, Assembler, Batch files, c , c++ , c#, Cascading Style Sheet, Fortran Fox Pro, html, Inform, Ini format, LaTex, Pascal, Perl, php, Python, Java, Java Script, SQL, TCL/Tk, Visual Basic, VbScript, XML, Unreal Tournament.
  • template scripting system based on ObjectPascal for adding new functions to AEdiX
  • built-in FTP client
  • other stuff.

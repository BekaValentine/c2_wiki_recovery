Efour Graph

It is called E4Graph but that doesn't make a wiki name because of the number.

See http://e4graph.sourceforge.net/ for details. It is an object oriented approach to data storage which sits on top of MetaKit.

It provides XmlDatabaseServers through linking in ExPat which provides XML parsing and generation. It stores the XML in a tree structure which mirrors the external structure of the data. -- JohnFletcher

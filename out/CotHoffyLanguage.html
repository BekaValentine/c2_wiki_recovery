Cot Hoffy Language

A new language has been created by the AFD Research Academy called Cot Hoffy". It's influenced by JavaScript, but uses pure HigherOrderFunctions (HOF's) for its root homoiconic structure (see HomoiconicFaq). "Everything is a HOF from head to toe", says Cutty Wardingham, language project lead.

Example:

// add a + b
function{function, function{(a)}{
function(function{function(b)}, function(
{{function.add()}}, function(), function())
}
//
// short-cut syntax:
//
f{f, f{(a)}{
f({f(b)}, (
{{f.add()}}, f(), f())
}

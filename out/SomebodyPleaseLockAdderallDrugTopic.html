There's an easy solution - ChangeTheCodeWord

On the file system of my wiki, when a page begins to get spammed, I simply use this unix command in my crontab:

cd wiki && chmod o-w SpamTargets `cat SpamTargets`

where SpamTargets is a wiki page collecting the page names that seem to be targeted. If anyone has access to the file system of this wiki, I recommend it. See also, and try to change my SpamTargets http://humu.ucsd.edu/cgi-bin/ChrisGarrod?SpamTargets -- ChrisGarrod

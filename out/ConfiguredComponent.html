Configured Component

The terminology used for a ComComponent that has been configured with certain attributes under ComPlus.

Some attributes include:

  • Transaction requirements
    • disabled
    • not supported
    • supported
    • required
    • requires new
  • RoleBasedSecurity
  • JustInTimeActivation (aka JITA)
  • ObjectPooling
  • Synchronization settings for ThreadNeutralApartment, SingleThreadedApartment

See also: ComponentObjectModel, ComPlus, TransactionalComponent

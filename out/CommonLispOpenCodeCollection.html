Common Lisp Open Code Collection

Common Lisp Open Code Collection (http://clocc.sourceforge.net/) is an archive of software for CommonLisp. It may not still be being updated.

There are also the following:

  • AnotherSystemDefinitionFacility (ASDF) http://common-lisp.net/project/asdf/
  • QuickLisp (http://www.quicklisp.org/)

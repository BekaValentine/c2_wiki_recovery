Idea Methodology

IDEA -- Intelligent Database Environment for Advanced Applications Methodology was defined as part of the IDEA Project.

What makes it interesting is that it is specific to the problem of building an OO Application on top of an existing RDBMS.

Developed was also the Chimera Language for support the design phase.

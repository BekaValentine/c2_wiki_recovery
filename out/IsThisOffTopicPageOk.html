Is This Off Topic Page Ok

If you are directed to this page from another page, it means there are some contentions on the merits of the source page, which is broadly understood to be OffTopic.

Please locate within this page the section where the source page is named, check out the arguments, add your own if needed, and give others a chance to form an opinion of the source page.

I suggest after a week or so of inactivity (merits discussions), then either the link to this page is removed, or the source page to be marked for delete, using DeletedUnlessDefended.

If the source page is subsequently removed, then I request interested parties to remove the discussion section in this page as well.

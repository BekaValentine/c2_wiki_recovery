Dynamic Web Twain

Dynamsoft Dynamic Web TWAIN is a web browser plugin for scanning documents from a TWAIN compatible device - scanner, digital camera or capture card. After acquiring images, you can edit and then upload them to web servers/databases.

Three editions:

ActiveX Edition works as an ActiveX control in IE (32-bit and 64-bit)
Plugin Edition works with Firefox, Chrome, Safari, and Opera on Windows.
Mac Edition works with Safari, Firefox, Chrome and Opera on Mac.

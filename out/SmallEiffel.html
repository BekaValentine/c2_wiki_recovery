Small Eiffel

As of 6 September 2002, the SmallEiffel project is renamed SmartEiffel. The current SmallEiffel web site is thus obsolete. All new developments can be found on the new SmartEiffel web site at http://SmartEiffel.loria.fr.

SmallEiffel is the GNU version of the EiffelLanguage. It is available from http://SmallEiffel.loria.fr/ .

A ready-to-go version of SmallEiffel for Windows that includes an syntax-based editor, an ANSI C compile and a generous set of libraries is available at http://www.elj.com/elj-win32/ .

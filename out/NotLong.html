Not Long

http://notlong.com/ is a simple web site by EricHammond which can create a short URL given a long one.

Other web sites which perform the same function include:

http://snipurl.com/

http://tinyurl.com/

http://urlizer.com/

http://shorl.com/

http://makeashorterlink.com/

http://shurl.org/

For a survey comparing features of these sites, see http://notlong.com/links/

Lava Language

Name for different unrelated languages.

  • Experimental object-oriented rapid application development language
See http://lavape.sourceforge.net/
  • An extension of the JavaLanguage by static and dynamic delegation and consultation.
See http://javalab.iai.uni-bonn.de/research/darwin/
  • A hardware description language based upon the HaskellLanguage.

JaWiki

  • is a small, servlet based wiki
  • runs using ApacheTomcat from version 4.1 with JDK from version 1.4.1
  • was written in 2004 by Thomas Schmiereck
  • is OpenSource and available on SourceForge: http://sourceforge.net/projects/jawiki/
  • has a description in German available from the author on this page: http://www.schmiereck.de/jawiki/user/main/JaWiki
  • needs no database; it uses file system based techniques to store the articles in ExtensibleMarkupLanguage files
  • supports versions of articles and provides a user management with a login

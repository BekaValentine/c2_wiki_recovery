Zeta OS should combine the strengths of BE OS and Windows OS. Take the SMP and media centrality of BEOS and combine that with the keyboard interface for Windows OS. Windows employs a consistent keyboard navigation system which allows users to Search Review and Organize information using nothing more than keyboard and shortcuts (if you don't believe me, figure out how to turn on mousekeys without using the mouse, then you'll begin to see the power of a Graphic Keyboard User Interface).

Doc++ (DocPlusPlus) is an automatic documentation generator. DOC++ extracts documentation directly from the C++ header or Java class files generating both LaTeX output for high quality hardcopies, and HTML output for sophisticated online browsing of your documentation.

http://www.zib.de/Visual/software/doc++/ (BrokenLink 20120322)

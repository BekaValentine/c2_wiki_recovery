An AntiPattern.

Make the project look bigger than it actually is so that more funding goes into the project.

It also looks better in the resume: It lasted longer, it had more people, therefore it was more difficult. All others seem toy projects compared to this one.

A simple way to do this is to add more people to a project that is already late.

Compare this with: The project I worked on had X million lines. MillionairesProjects?

A simple way to do this is to use CopyAndPaste.

See WhatIsaProject.

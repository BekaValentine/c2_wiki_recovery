Jazz is an IBM Rational project to build a scalable, extensible team collaboration platform for integrating work across the phases of the development lifecycle. We believe Jazz will help teams build software more effectively while making the software development activity more productive and enjoyable.

Web Site: https://jazz.net/pub/index.jsp

The Jazz Source Control (SCM) component is a light weight SCM system built on the Jazz platform. More information about the SCM features in Jazz can be found at https://jazz.net/jazz/web/projects/Jazz%20Project#action=com.ibm.team.dashboard.viewDashboard&team=Jazz%20Development/Source%20Control.

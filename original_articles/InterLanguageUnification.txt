InterLanguageUnification is an OpenSource ObjectRequestBroker from XeroxParc.

ftp://ftp.parc.xerox.com/pub/ilu/ilu.html

----
The W3 people working on HTTP-NG seem pretty keen on it:

''ILU is a very promising technology for tying the web to object oriented programming. The more I see, the more I am convinced it will work. Here's why.''  http://www.w3.org/OOP/WhyILU (written in 1995)

CORBA defines a standard transport and wire protocol (IIOP/TCP), whereas ILU defines an API and set of concepts and supports multiple transports.

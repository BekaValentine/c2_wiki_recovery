Draco.NET provides ContinuousIntegration using NantTool or VisualStudioDotNet.  It runs as a Windows Service. 

'''Project Website:'''  http://draconet.sourceforge.net

'''SourceForge Project Summary:''' http://sourceforge.net/projects/draconet

'''Supported VCS systems:'''
	* CVS (ConcurrentVersionsSystem)
	* PvcsVersionControl
	* SubVersion
	* VisualSourceSafe

'''Supported Build systems'''
	* VSNet 2002
	* Nant

Doesn't seem to support VSNet 2003 as build tool.

A GnuGeneralPublicLicense toolkit for creating applications that use the Microsoft Messaging ApplicationProgrammingInterface.

The toolkit and other resources are available at http://www.sourcextreme.com/resources.html

The code started out as part of a project to build an OpenSource MAPI transport and MAPI message store. The code has evolved, predominantly using TestDrivenDevelopment.

This toolkit is FreeSoftware and OpenSource. It was originally authored by JasonNocks and is provided by SourceXtreme, Inc. Commercial support is also available. Contact mailto:info@sourcextreme.com. Or, visit http://www.sourcextreme.com/ for more information about SourceXtreme.

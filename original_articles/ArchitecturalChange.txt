Can an architecture of a system change? 

''Experiences from the Architectural Change Process''

http://citeseer.ist.psu.edu/644078.html
----
'''No.''' Architecture is the base, irreducible, and immutable set of rules defining the operation of a system. Once those rules are set that's it. If you want to change the architecture then you need to define a different system.

ikiwiki (a palindrome) is a wiki compiler. It converts wiki-format pages into HTML pages. It has an abstract interface for page revision storage, allowing many types of VersionControl or database to be used. It is written in PerlLanguage.

ikiwiki has a plugin architecture, and supports many types of WikiSyntax, including MarkDown and ReStructuredText. It also has a module for creating blogs, RSS feeds, bug lists, and other types of listable content.

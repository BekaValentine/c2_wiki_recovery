	XOTcl:	Extended Object Tcl (TCL = ToolCommandLanguage)
An extension to the ToolCommandLanguage that provides sophisticated OO functionality with a high level of introspection and dynamics. Influenced by the CLOS. Some features:
	* Method filtering
	* Meta-classes
	* Dynamic class relationships
	* Multiple inheritance
	* Mixins
	* Assertions
	* Full read/write introspection
For more information, see http://www.xotcl.org/
----
XOTcl is available in a Tcl package called ActiveTcl.

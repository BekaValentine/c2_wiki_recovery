Diri is a minimalist wiki engine written in under 200 PlanNineFromUserSpace rc shell. It can run as a CGI and uses a plain filesystem for storage.

Its name comes from its use of directories to organize wiki pages.

Some included features are:
* Markdown formatting
* Edit preview
* Password protection
* Hierarchical organization and navigation of pages

The source and some minimal documentation can be found at http://repo.cat-v.org/diri/

The original version of diri was implemented by garbeam, later improvements were done by Kris Maglione and it is currently maintained by uriel.

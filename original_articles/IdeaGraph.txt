http://ideagraph.net/ (BrokenLink 2008/11/21)

A tool (in mid-beta) that uses SemanticWeb technologies to make an seriously powerful knowledge manager. Novel user interface elements make the tool user-friendly. The application as a whole will be commercial, though virtually all reusable parts will be released open source (Java).

What does it do?

''It seems to be some sort of MindMap editor, plus the ability to import stuff from Web pages and RSS feeds, and generate Web pages and RSS feeds. Anything else?''

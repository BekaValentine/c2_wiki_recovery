Java Card is SUN Microsystem's Java technology platform for application development on smart cards.

There are three specifications by Sun which define on-card Java:
- Java Card Virtual Machine Specification
- Java Card Runtime Environment Specification
- Java Card Application Programming Interface Specification

Further details can be found at http://java.sun.com/products/javacard

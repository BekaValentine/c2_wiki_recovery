''OO++: Exploring the Multiparadigm Shift'' Seth James Nielson and Charles D. Knutson

* http://sequoia.cs.byu.edu/lab/files/pubs/Nielson2004a.pdf 

This paper discusses the way in which programming languages and programming language paradigms have evolved. It surveys the languages which have moved from being object oriented to contain other paradigms as well.

Examples are developed using PythonLanguage.

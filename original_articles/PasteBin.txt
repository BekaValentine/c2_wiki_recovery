A WebSite with a textarea to paste text. The text is stored and given a URL. Visiting the URL displays the text. Code is usually syntax colored and formatted according to language. Useful for pasting into IRC (paste the url, not the entire code) or as a ProgrammersNotebook.

Many pastebins exist. Here are a few:
* http://pastebin.com
* http://paste.lisp.org
* http://paste2.org
* http://codepad.org
* http://pastie.org

Stikked http://github.com/claudehohl/Stikked is a PHP/MySQL pastebin you can install on your own server.

GitHub gists are a form of pastebin, with VersionControl

Personal Python Qwiki with MindMap feature is a quick and easy to setup wiki system. Its sources are only about 36KB but it offers:
* of-course WikiLinks and Page editing in the WebBrowser
* Whitespaces in WikiLinks
* formatting(bold, italic, underlined)
* tables
* external links
* reference links
* including images from a local path
* a web based menu
* transfer between flat files and WikiTree in both directions
* wiki to one document 
* GUI (freemind) for page order
the possibility to create xml-files to be used in FreeMind
It needs:
* FreeMind including Java
* Python
* a file system :-)
This project is inspired by QwikiWiki 

Download and further details at:
http://sourceforge.net/projects/ppqwiki/

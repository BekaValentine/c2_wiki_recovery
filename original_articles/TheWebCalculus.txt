The web-calculus merges the RestArchitecturalStyle underlying the WWW with a CapabilitySecurityModel. The resulting model and protocol enables secure, distributed applications atop the existing WWW infrastructure. Existing CGI environments can be used to produce server applications. Existing web browsers are fully functional clients for web-calculus applications.

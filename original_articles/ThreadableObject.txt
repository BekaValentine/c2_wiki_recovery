Another MicroArchitecture production.

This is an object which can be executed. As opposed to an object which is only state (a StateObject).

In the OO world this kind of object is assumed to exist. There are various subtypes including those objects which can be multithreaded (reeentrant), or which enforce single threading (synchronized).

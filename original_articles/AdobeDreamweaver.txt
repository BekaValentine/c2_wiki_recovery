Dreamweaver is a WhatYouSeeIsWhatYouGet HTML editor from Adobe (formally MacroMedia, prior to 2005).  It features support for a number of web page types - (X)HyperTextMarkupLanguage, ActiveServerPages, ColdFusion, AspDotNet, JavaServerPages, CascadingStyleSheets, etc.

http://www.adobe.com/products/dreamweaver

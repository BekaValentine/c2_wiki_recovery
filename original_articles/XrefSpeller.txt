
------

Xref-Speller has been renamed to Xrefactory.

------

Xrefactory is plug-in to Emacs/XEmacs editors. Originally it provided C code completion and source browsing, later it evolved to a refactoring tool. After apparition of Java language, a Java module was implemented and was 
discontinued after apparition of Java 1.5. Currently Xrefactory supports C and C++ on a professional level. It provides several refactorings like renaming of variable, function argument insert, delete, reorder; ExtractMethod. The home page is at http://www.xref-tech.com/xrefactory .

Setting up the KawaIde to run JavaUnit tests.

 Add junit.jar to your classpath.
 Open Kawa.
 Select Commands from the Customize menu.
 Click the New button on the Custom Commands dialog.
 Type "JUnit" in the Menu Name text field.
 Type "java.exe" in the Command text field.
 Type "junit.swingui.TestRunner  <Class of File>" in the Arguments text field.
 Type "<File Directory>" in the Startup Directory text field.
 Check the "Set projects classpath before running the command" check box.
 Choose "Redirect Output only to Ouput Window" from the Input/Output chooser.
 Click the Options tab.
 Check the "Add command to File Item right button menu in project tree" check box.
 Check the "Add command to Project Item right button menu in project tree" check box.
 Check the "Add command to right button menu in editor" check box.

UniWakka is a PHP based WikiEngine derived from WakkaWiki.

http://uniwakka.sourceforge.net

Core features:
* Formulas can be inserted as LaTex, ASCIIMathML, and plain MathMl
* Chemical structures can be visualized with the Jmol Applet
* WikiFarms installations
* Multi-language support
* RichSiteSummary
* Table of Contents and footnotes
* BibTex import and export
* LaTex export
* LaTex-like citations
* OpenOffice export
* OpenOffice export filter, that lets you use Writer to create content for the wiki and then export it in UniWakka format
* Calendar
* Basic spell-checking support

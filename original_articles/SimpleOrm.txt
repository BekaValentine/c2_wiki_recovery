Overview
SimpleORM is new Java Object Relational Mapping open source project (Apache style licence).  It provides a very simple and easy way to implement object/relational mapping on top of JDBC at low cost and low overhead.  Not even an XML file to configure!

----
Superb tool, database agnostic and allows to debug easily.
